# SEO Content Cluster Generator - Dental Clinic Gauteng

## Project Overview

This project implements a complete **SEO Content Cluster Strategy** for a dental clinic business in Gauteng, South Africa. It follows professional SEO agency standards for generating ranking-focused content that drives organic traffic and generates leads.

## Business Profile

- **Business Type:** Dental Clinic / Dental Care Services
- **Location:** Gauteng, South Africa (Johannesburg, Pretoria, Sandton)
- **Target Audience:** Local residents seeking dental services
- **Primary Goal:** Increase organic visibility and convert searchers into patients

## Content Cluster Strategy

### Pillar Content (Main Topic)

**"Complete Dental Care in Gauteng"** - A comprehensive guide covering all aspects of dental health services available in Gauteng.

### Supporting Content (Cluster Topics)

1. **Dental Implants Johannesburg** - Service-specific, high-intent keyword
2. **Teeth Whitening Guide** - Cosmetic procedure focus
3. **Emergency Dentist Gauteng** - Urgent care search intent
4. **Braces Cost South Africa** - Price comparison content
5. **How to Choose the Best Dentist** - Decision-making guide

## Prompt System Architecture

The project uses a **multi-tier prompt system**:

1. **Master Prompt** - Sets context, business profile, and goals
2. **Pillar Blog Prompt** - Generates comprehensive main content
3. **Supporting Blog Prompt** - Creates focused cluster content
4. **Local SEO Modifier** - Adds location-specific optimization

Each prompt can be reused for different businesses by modifying the parameters.

## How to Use This System

### For a New Business:

1. Copy `prompts/` folder structure
2. Modify business profile in MASTER_PROMPT_SYSTEM.md
3. Update location in local_seo_modifier.md
4. Generate new content using the prompts
5. Follow seo-implementation/ checklist

### For Content Generation:

1. Start with Master Prompt for context
2. Use Pillar Blog Prompt for main content
3. Generate 3-5 supporting blogs with Supporting Blog Prompt
4. Apply Local SEO Modifier to all content
5. Complete SEO Checklist before publishing
