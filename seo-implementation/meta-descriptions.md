## Meta Description Templates

### Service Page Template

```
[Primary Keyword] in [Location]: [Benefit].
[Supporting detail about service].
[CTA].
```

**Example:**

```
Dental implants in Johannesburg: Restore your smile.
Permanent tooth replacement with natural-looking results.
Book your consultation today.
```

### Informational Template

```
Learn about [topic] in [Location].
[Key benefit or value proposition].
[CTA].
```

**Example:**

```
Learn about professional teeth whitening in Gauteng.
Safe, effective treatments for a brighter smile.
Find whitening dentists near you.
```

### Emergency Template

```
Emergency [service] in [Location].
[Immediate benefit or solution].
[Urgent CTA].
```

**Example:**

```
Emergency dentist in Gauteng now available.
Immediate relief for toothache and dental trauma.
Call now for urgent care.
```

### Comparison/Guide Template

```
[Topic] in [Location]: [What readers will learn].
[Supporting detail].
[CTA].
```

**Example:**

```
Braces cost in South Africa explained.
Compare metal, ceramic, and Invisalign pricing.
Get your orthodontic consultation today.
```

## Meta Description Best Practices

### Do:

✅ Include primary keyword near the beginning
✅ Keep within 150-160 characters
✅ Include a clear call-to-action
✅ Make it compelling and click-worthy
✅ Add location where relevant
✅ Use active voice
✅ Match the page content accurately

### Don't:

❌ Exceed 160 characters (will truncate)
❌ Use keyword stuffing
❌ Make misleading claims
❌ Use generic, boring language
❌ Forget the CTA
❌ Duplicate meta descriptions across pages
❌ Use quotes (can truncate in search results)

## Character Count Reference

| Length  | Status     | Impact               |
| ------- | ---------- | -------------------- |
| < 120   | Too short  | Missing information  |
| 120-150 | Acceptable | Room for improvement |
| 150-160 | Optimal    | Best practice        |
| > 160   | Too long   | Will truncate        |
