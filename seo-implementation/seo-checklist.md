## Pre-Publication Checklist

### Technical SEO

| Task                       | Status | Notes                                    |
| -------------------------- | ------ | ---------------------------------------- |
| URL structure optimized    | ☐      | Include primary keyword, readable format |
| HTTPS enabled              | ☐      | Verify SSL certificate                   |
| XML sitemap updated        | ☐      | Add new pages to sitemap                 |
| Robots.txt allows crawling | ☐      | Verify no blocks on content              |
| Page load speed optimized  | ☐      | Target under 3 seconds                   |
| Mobile responsive design   | ☐      | Test on multiple devices                 |
| Core Web Vitals pass       | ☐      | LCP, FID, CLS all green                  |
| Canonical tags set         | ☐      | Prevent duplicate content                |

### On-Page SEO Elements

| Task                                       | Status | Notes                      |
| ------------------------------------------ | ------ | -------------------------- |
| Title tag optimized (50-60 chars)          | ☐      | Include primary keyword    |
| Meta description optimized (150-160 chars) | ☐      | Include CTA                |
| H1 tag present and unique                  | ☐      | One per page, with keyword |
| H2-H6 structure logical                    | ☐      | Proper heading hierarchy   |
| Primary keyword in first 100 words         | ☐      | Natural placement          |
| Keywords in subheadings                    | ☐      | 2-3 H2/H3 headings         |
| LSI keywords throughout content            | ☐      | 5-10 semantic variations   |
| Internal links added (3-5)                 | ☐      | Link to cluster content    |
| External links to authority sites          | ☐      | 2-3 credible sources       |
| Image alt text optimized                   | ☐      | Descriptive, keyword-rich  |

### Content Quality

| Task                          | Status | Notes                                    |
| ----------------------------- | ------ | ---------------------------------------- |
| Word count meets target       | ☐      | Pillar: 2500-4000, Supporting: 1200-2000 |
| Prose-first approach (80%+)   | ☐      | Minimize lists, use paragraphs           |
| Paragraphs 2-4 sentences each | ☐      | Scannable, readable format               |
| No keyword stuffing           | ☐      | Natural language only                    |
| Content accuracy verified     | ☐      | Facts, statistics current                |
| No plagiarism                 | ☐      | Original content only                    |
| Readability grade 6-8         | ☐      | Accessible to general audience           |
| Clear value proposition       | ☐      | Reader knows benefit                     |

### Local SEO

| Task                            | Status | Notes                        |
| ------------------------------- | ------ | ---------------------------- |
| Location in title tag           | ☐      | City/area included           |
| NAP consistent across site      | ☐      | Name, Address, Phone         |
| Google Business Profile aligned | ☐      | Match service names          |
| Location mentioned naturally    | ☐      | 5-8 times in content         |
| Neighborhoods referenced        | ☐      | Local areas served           |
| Map embed if applicable         | ☐      | On contact/location pages    |
| Local schema markup             | ☐      | LocalBusiness/Dentist schema |
| Location pages optimized        | ☐      | Each service area has page   |

### User Experience

| Task                       | Status | Notes                       |
| -------------------------- | ------ | --------------------------- |
| Clear navigation           | ☐      | Easy to find content        |
| CTAs prominent             | ☐      | Clear call-to-action        |
| Trust signals present      | ☐      | Reviews, credentials, certs |
| Contact info visible       | ☐      | Phone, address, hours       |
| Fast loading images        | ☐      | Optimized file sizes        |
| No intrusive interstitials | ☐      | Good mobile experience      |
| Content scannable          | ☐      | Subheadings, short sections |
| Mobile-friendly formatting | ☐      | Readable without zoom       |
